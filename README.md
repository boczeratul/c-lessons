# C++ lessons

These lessons are designed for a cute girl named Arty, which has nothing to Moriarty.

### Clone the repository
```bash
$ git clone git@github.com:boczeratul/c-lessons.git
$ cd c-lessons
```

Edit and Compile
```bash
$ g++ exercise.cpp 
# or other files
```

Execute
```bash
$ ./a.out
```
